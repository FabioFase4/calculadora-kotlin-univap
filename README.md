# Calculadora Android (Kotlin)

Projeto desenvolvido para a disciplina de Programação no curso de Informática da **UNIVAP**.

## 🚀 Funcionalidades
- Operações matemáticas básicas (+, -, *, /).
- Cálculo acumulado (permite fazer 2+2+2...).
- Tratamento de erro para divisão por zero.
- Cores dinâmicas para feedback visual.

## 🛠️ Tecnologias
- Linguagem: Kotlin
- IDE: Android Studio
- Componentes: TextView, Button, ConstraintLayout.
