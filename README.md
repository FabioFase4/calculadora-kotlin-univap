# 🐤 Calculadora Funcional 
# Autores:
- Fábio Silva de Lima
- Guilherme Henrique Gonzales Nascimento Barreto

# :accessibility: Recursos Usados 
- Linguagem Kotlin
- Android Studio (IDE)

# 🚀 Objetivos 
- Simular uma Calculadora com as 4 Operações Básicas da Matemática
- ➕ ➖ ✖️ ➗
- Uso de Valores Inteiros 👼

# ⛽ Funcionalidades 
- Digitar Números Inteiros com os Algarismos de 0 a 9 ⏰
- Exclusividade para Celulares Android e Emuladores desses Celulares 🪕
- Digitar alguma Operação Básica 🏖️
- Mostrar o Resultado 🚑

# Funcionalidades Futuras 🏛️
- Armazenamento de Operações usando Estruturas de Dados 🦋
- Inclusão de Mais Operações 🚡
- Inclusão de Valores Decimais com Vírgula ⚓
